History of the language

A Language That Yields Medical Process HAndling is what lympha stands for. Like every language, it is not only a way to express yourself. It also works the other way, since a language gives tools for building new ideas.

In early 2013 Rickard Verner Hultgren created the project. First, it was a script for making runnable flowcharts in python using the ruffus library.In late 2013 the development of the language began as an interpreter on top of the script. In late 2014 the project was completely rewritten.
